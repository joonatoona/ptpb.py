# Usage:

Upload:

```
echo 'Upload me!' | pbc
```

or

```
pbc 'Upload me!'
```

or

```
pbc /path/to/file.whatever
```

Update:

```
echo 'Update me!' | pbu <Paste URL>

```

or

```
echo 'Update me!' | pbu <Paste UUID>
```

Can be used the same way as `pbc`

Delete:

```
pbc <Paste URL>
```

or

```
pbc <Paste UUID>
```

---

All paste UUIDs are stored in ~/.ptpbstore.json
